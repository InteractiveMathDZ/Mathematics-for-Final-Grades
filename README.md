## 📜 ميثاق المشروع (The Project Charter)
بعد ضبط الأدوات وتوفر التوطين، نعتمد هذا التقسيم كدستور تقني وبيداغوجي للمشروع، لضمان أعلى درجات الإتقان ومنع اختلاط الملفات:

### 🏗️ أولاً: هيكلة المجلدات (المجالات الستة)
يتم تنظيم الدروس في مجلدات كبرى تعكس المخطط السنوي الرسمي:
1. **التحليل:** (النهايات، الاستمرارية، الاشتقاق، الدوال الأسية واللوغاريتمية، التزايد المقارن).
2. **المتتاليات العددية:** (البرهان بالتراجع، التقارب، والمتتاليات التراجعية).
3. **الحساب التكاملي:** (الدوال الأصلية والتكامل وحساب المساحات).
4. **الجبر والحساب:** (القسمة، الموافقات، والأعداد الأولية).
5. **الأعداد المركبة والهندسة:** (المركبة، التحويلات، والهندسة الفضائية).
6. **الاحتمالات:** (الاحتمالات الشرطية وقوانين الاحتمال).

### 🛠️ ثانياً: ضوابط التنفيذ التقني
* **التوحيد:** الالتزام بـ `style.css` الموحد في مجلد `assets`.
* **التناسق:** اعتماد وسم **"تطبيق مباشر"** لجميع الفقرات بلا استثناء.
* **الملاحة:** كل صفحة درس أو تمرين تنتهي بـ (زر المتابعة ⬅️) و (زر الرجوع ➡️).
* **الاستقلالية:** لكل درس أو تمرين استزادة ملفه الخاص لمنع تراكم الأخطاء.
