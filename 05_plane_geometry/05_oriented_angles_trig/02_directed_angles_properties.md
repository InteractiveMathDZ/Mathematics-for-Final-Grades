  - h3: "1. علاقة شال للزوايا الموجهة"
    id: "chasles-relation-angles"

  - parags:
      - rtl: "تسمح علاقة شال للزوايا الموجهة بدمج الدوران المتتالي عبر إدخال شعاع وسيط $\\vec{w}$ بين الشعاعين $\\vec{u}$ و $\\vec{v}$. قيس الدوران الإجمالي يساوي مجموع القيسين الجزئيين (ترديد $2\\pi$):"
      - center: "$$(\\vec{u}, \\vec{v}) = (\\vec{u}, \\vec{w}) + (\\vec{w}, \\vec{v}) + 2k\\pi \\quad \\text{where } k \\in \\mathbb{Z}$$"
      - rtl: "<strong>الفائدة البيداغوجية:</strong> هذه الخاصية هي الأداة الأقوى لتفكيك الزوايا المركبة داخل المثلثات والأشكال الهندسية لحساب القياسات الإجمالية."

  - h3: "2. الأثر الإشاري عند عكس الأشعة والترتيب"
    id: "sign-effects"

  - parags:
      - rtl: "تتميز الزوايا الموجهة بحس حركي صارم؛ أي تغيير في ترتيب الأشعة أو اتجاهها يغير القيس وفق القوانين الجبرية الحاكمة التالية:"

  - tableLines:
      dir: rtl
      lines:
        - - th: "التحويل الشعاعي للزاوية"
          - th: "الصياغة الجبرية الدقيقة"
          - th: "التفسير الهندسي والحركي"
        - - th: "عكس ترتيب الأشعة"
          - td: "$(\\vec{v}, \\vec{u}) = -(\\vec{u}, \\vec{v}) + 2k\\pi$"
          - td: "قلب الترتيب يعني الدوران في الاتجاه المعاكس حتماً (تغير الإشارة)."
        - - th: "معاكس الشعاع الثاني"
          - td: "$(\\vec{u}, -\\vec{v}) = (\\vec{u}, \\vec{v}) + \\pi + 2k\\pi$"
          - td: "عكس اتجاه الشعاع الثاني يضيف نصف دورة ($\\pi$) للمسار الدوراني."
        - - th: "معاكس الشعاع الأول"
          - td: "$(-\\vec{u}, \\vec{v}) = (\\vec{u}, \\vec{v}) + \\pi + 2k\\pi$"
          - td: "عكس اتجاه الشعاع الأول يضيف كذلك نصف دورة ($\\pi$)."
        - - th: "معاكس الشعاعين معاً"
          - td: "$(-\\vec{u}, -\\vec{v}) = (\\vec{u}, \\vec{v}) + 2k\\pi$"
          - td: "عكس اتجاه كلا الشعاعين يلغي أثر الدوران الإضافي ويعود القيس لأصله تماماً."

  - h3: "3. شرطا الاستقامية والتعامد بلغة الزوايا"
    id: "geometric-conditions-angles"

  - tableLines:
      dir: rtl
      lines:
        - - th: "الوضعية الهندسة المستهدفة"
          - th: "الشرط الحاكم لقيس الزاوية $(\\vec{u}, \\vec{v})$"
          - th: "المدلول البصري"
        - - th: "الارتباط الخطي (الاستقامية/التوازي)"
          - td: "$(\\vec{u}, \\vec{v}) = 0 \\quad \\text{or } \\quad (\\vec{u}, \\vec{v}) = \\pi$"
          - td: "إما نفس الاتجاه ($0$) أو اتجاهان متعاكسان ($\\pi$)."
        - - th: "التعامد القطبي"
          - td: "$(\\vec{u}, \\vec{v}) = \\frac{\\pi}{2} \\quad \\text{or } \\quad (\\vec{u}, \\vec{v}) = -\\frac{\\pi}{2}$"
          - td: "زاوية قائمة تماماً في الاتجاه المباشر أو غير المباشر."

  - h3: "4. مثال تطبيقي (تفكيك عبارة زاوية معقدة لشعاعين متعاكسين)"
    id: "angles-prop-ex"

  - parags:
      - rtl: "علماً أن قيس الزاوية الموجهة $(\\vec{u}, \\vec{v}) = \\frac{\\pi}{4}$، احسب القياس الرئيسي للزاوية الموجهة التالية: $\\theta = (-\\vec{v}, \\vec{u})$."

  - tableLines:
      dir: rtl
      lines:
        - - th: "المرحلة"
          - th: "التطبيق الجبري للخواص"
          - th: "النتيجة والتبرير"
        - - th: "1. معالجة الإشارة السلبية"
          - td: "$(-\\vec{v}, \\vec{u}) = (\\vec{v}, \\vec{u}) + \\pi$"
          - td: "عكس اتجاه الشعاع الأول يضيف نصف دورة ($\\pi$)."
        - - th: "2. قلب ترتيب الأشعة"
          - td: "$(\\vec{v}, \\vec{u}) = -(\\vec{u}, \\vec{v}) = -\\frac{\\pi}{4}$"
          - td: "قلب الترتيب يعكس إشارة القيس الأصلي المعطى."
        - - th: "3. التجميع الحسابي"
          - td: "$\\theta = -\\frac{\\pi}{4} + \\pi = -\\frac{\\pi}{4} + \\frac{4\\pi}{4} = \\frac{3\\pi}{4}$"
          - td: "توحيد المقامات وحساب القيس النهائي."
        - - th: "4. فحص المعيار الرئيسي"
          - td: "$\\frac{3\\pi}{4} \\in ]-\\pi ; \\pi]$"
          - td: "بما أن النتيجة محصورة في المجال الحاكم، فهي القياس الرئيسي القطعي."
