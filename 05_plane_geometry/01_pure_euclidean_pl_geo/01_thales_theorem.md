  - parags:
      - rtl: "لتطبيق مبرهنة طاليس وحساب الأطوال، وجب تحقق شرطين أساسيين معاً:"

  - tableLines:
      dir: rtl
      lines:
        - - th: "الشرط"
          - th: "التفسير الهندسي"
          - th: "النتيجة المباشرة"
        - - th: "1. الاستقامية"
          - td: "النقاط $A, M, B$ في استقامية، والنقاط $A, N, C$ في استقامية بنفس الترتيب."
          - td: "تشكيل تشابه بنيوي للمثلثين الناتجين."
        - - th: "2. التوازي"
          - td: "المستقيمان $(MN)$ و $(BC)$ متوازيان كلياً: $(MN) // (BC)$."
          - td: "جواز صياغة نسب التناسبية الثلاث."

  - h3: "2. الدستور الهندسي (النسب الثلاث)"
    id: "thales-formulas"

  - parags:
      - rtl: "إذا تحقق الشرطان، فإن أطوال أضلاع المثلثين $AMN$ و $ABC$ تكون متناسبة ونعبر عنها رياضيا ب:"
      - center: "$$\\frac{AM}{AB} = \\frac{AN}{AC} = \\frac{MN}{BC}$$"

  - h3: "3. المبرهنة العكسية لطاليس (إثبات التوازي)"
    id: "thales-inverse"

  - parags:
      - rtl: "نستعمل المبرهنة العكسية عندما تكون الأطوال معلومة ونريد اثبات شرط التوازي:"
      - rtl: "- إذا كانت النقاط $A, M, B$ و $A, N, C$ على استقامية وبنفس الترتيب."
      - rtl: "- وتطابقت النسبتان: $\\frac{AM}{AB} = \\frac{AN}{AC}$."
      - rtl: "<strong>النتيجة:</strong> المستقيمان $(MN)$ و $(BC)$ متوازيان حتماً."

  - h3: "4. مثال تطبيقي (حساب طول مجهول)"
    id: "thales-ex"

  - parags:
      - rtl: "في مثلث $ABC$، لدينا $(MN) // (BC)$ حيث $AB = 6$، $AM = 2$، و $BC = 9$. احسب الطول $MN$."

  - tableLines:
      dir: rtl
      lines:
        - - th: "المرحلة"
          - th: "التعويض والحساب"
          - th: "النتيجة"
        - - th: "صياغة النسب"
          - td: "$\\frac{AM}{AB} = \\frac{MN}{BC} \\implies \\frac{2}{6} = \\frac{MN}{9}$"
          - td: "عزل المجهول"
        - - th: "الرابع المتناسب"
          - td: "$MN = \\frac{2 \\times 9}{6} = \\frac{18}{6}$"
          - td: "$MN = 3$"
