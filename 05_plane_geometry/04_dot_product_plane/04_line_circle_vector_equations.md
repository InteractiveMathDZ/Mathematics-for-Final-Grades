  - h3: "1. المستقيم المعرّف بنقطة وشعاع ناظمي"
    id: "normal-vector-line"

  - parags:
      - rtl: "نقول عن شعاع غير معدوم $\\vec{n}=\\binom{a}{b}$ أنه <strong>شعاع ناظمي</strong> للمستقيم $(\\Delta)$ إذا كان حامله عمودياً على هذا المستقيم. لتكن $A(x_A, y_A)$ نقطة معلومة من $(\\Delta)$، تنتمي النقطة $M(x, y)$ للمستقيم إذا وفقط إذا كان الشعاع $\\vec{AM}$ عمودياً على الشعاع الناظمي $\\vec{n}$:"
      - center: "$$\\vec{AM} \\cdot \\vec{n} = 0 \\iff a(x - x_A) + b(y - y_A) = 0$$"
      - rtl: "<strong>النتيجة:</strong> بنشر وتبسيط العبارة، نجد المعادلة الديكارتية للمستقيم من الشكل: $ax + by + c = 0$، حيث مركبات الشعاع الناظمي هي معاملات $x$ و $y$."

  - tableLines:
      dir: rtl
      lines:
        - - th: "المعادلة الديكارتية للمستقيم"
          - th: "الشعاع الناظمي $\\vec{n}$"
          - th: "شعاع التوجيه $\\vec{u}$"
        - - th: "$ax + by + c = 0$"
          - td: "$\\vec{n} = \\binom{a}{b}$"
          - td: "$\\vec{u} = \\binom{-b}{a}$"

  - h3: "2. المعادلة الديكارتية لدائرة معرفة بمركزها ونصف قطرها"
    id: "circle-equation-center"

  - parags:
      - rtl: "الدائرة $(\\mathcal{C})$ التي مركزها النقطة الثابتة $\\Omega(x_\\Omega, y_\\Omega)$ ونصف قطرها هو العدد الحقيقي الموجب $R$، هي مجموعة النقط $M(x, y)$ التي تحقق المسافة: $\\Omega M = R$. بتربيع الطرفين وتطبيق قانون المسافة بين نقطتين، نجد معادلة الدائرة:"
      - center: "$$(x - x_\\Omega)^2 + (y - y_\\Omega)^2 = R^2$$"
      - rtl: "<strong>ملاحظة:</strong> بنشر المتطابقات الشهيرة في العبارة السابقة، تؤول المعادلة إلى الشكل: $x^2 + y^2 - 2x_\\Omega x - 2y_\\Omega y + c = 0$."

  - h3: "3. معادلة دائرة معرفة بأحد أقطارها"
    id: "circle-equation-diameter"

  - parags:
      - rtl: "إذا كانت الدائرة $(\\mathcal{C})$ معرفة بقطرها $[AB]$ حيث $A$ و $B$ نقطتان معلومتان، فإن كل نقطة $M(x, y)$ من الدائرة تحقق شرط تعامد الشعاعين $\\vec{MA}$ و $\\vec{MB}$:"
      - center: "$$\\vec{MA} \\cdot \\vec{MB} = 0 \\iff (x_A - x)(x_B - x) + (y_A - y)(y_B - y) = 0$$"

  - h3: "4. مثال تطبيقي (تعيين معادلة مستقيم ومعادلة دائرة)"
    id: "line-circle-ex"

  - parags:
      - rtl: "أوجد المعادلة الديكارتية للمستقيم $(\\Delta)$ المار بالنقطة $A(1, 4)$ والذي يقبل $\\vec{n}=\\binom{3}{-2}$ كشعاع ناظمي له. ثم أوجد معادلة الدائرة التي مركزها $\\Omega(2, -1)$ ونصف قطرها $R=5$."

  - tableLines:
      dir: rtl
      lines:
        - - th: "المطلوب"
          - th: "خطوات التعويض والحساب"
          - th: "معادلة المنحنى"
        - - th: "معادلة المستقيم $(\\Delta)$"
          - td: "بما أن $\\vec{n}=\\binom{3}{-2}$ شعاع ناظمي، فإن المعادلة تكتب على الشكل: $3x - 2y + c = 0$.<br>نعوض إحداثيات النقطة $A(1,4)$ لتعيين قيمة $c$: $3(1) - 2(4) + c = 0 \\implies 3 - 8 + c = 0 \\implies c = 5$."
          - td: "$(\\Delta): 3x - 2y + 5 = 0$"
        - - th: "معادلة الدائرة $(\\mathcal{C})$"
          - td: "نعوض مباشرة في قانون المركز ونصف القطر:<br>$(x - 2)^2 + (y - (-1))^2 = 5^2$<br>$(x - 2)^2 + (y + 1)^2 = 25$."
          - td: "$(\\mathcal{C}): (x - 2)^2 + (y + 1)^2 = 25$"
