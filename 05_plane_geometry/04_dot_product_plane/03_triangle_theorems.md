elements:
  - h3: "1. مبرهنة الكاشي"
    id: "al-kashi-theorem"

  - parags:
      - rtl: "في أي مثلث $ABC$ كيفي، نرمز لأطوال الأضلاع المقابلة للرؤوس $A$، $B$ و $C$ بالرموز $a$، $b$ و $c$ على الترتيب (حيث $BC=a$، $AC=b$، $AB=c$). تسمح مبرهنة الكاشي (قانون جيب التمام) بحساب طول ضلع مجهول بمعرفة الضلعين الآخرين وقيس الزاوية المحصورة بينهما وفق العلاقات التالية:"
      - center: "$$a^2 = b^2 + c^2 - 2bc \\cos(\\hat{A})$$"
      - center: "$$b^2 = a^2 + c^2 - 2ac \\cos(\\hat{B})$$"
      - center: "$$c^2 = a^2 + b^2 - 2ab \\cos(\\hat{C})$$"
      - rtl: "<strong>ملاحظة منهجية:</strong> إذا كانت إحدى الزوايا قائمة (مثلاً $\\cos(90^\\circ)=0$)، تؤول العبارة مباشرة إلى مبرهنة فيثاغورس ($a^2 = b^2 + c^2$)."

  - h3: "2. مبرهنة المتوسط"
    id: "median-theorem"

  - parags:
      - rtl: "ليكن $ABC$ مثلثاً كيفياً، و $I$ منتصف الضلع $[BC]$. تسمى القطعة المستقيمة $[AI]$ بالمتوسط المتعلق بالضلع $[BC]$. تسمح مبرهنة المتوسط بحساب طول هذا المتوسط انطلاقاً من أطوال أضلاع المثلث الثلاثة وفق العلاقة التالية:"
      - center: "$$AB^2 + AC^2 = 2AI^2 + \\frac{BC^2}{2}$$"
      - rtl: "<strong>البرهان الشعاعي:</strong> تنتج هذه المبرهنة بتطبيق علاقة شال وتطوير المربع السلمي للعبارتين $\\vec{AB} = \\vec{AI} + \\vec{IB}$ و $\\vec{AC} = \\vec{AI} + \\vec{IC}$، مع الاستعانة بشرط المنتصف ($\\vec{IB} + \\vec{IC} = \\vec{0}$)."

  - h3: "4. مثال تطبيقي (حساب طول ضلع وطول متوسط)"
    id: "triangle-theorems-ex"

  - parags:
      - rtl: "ليكن $ABC$ مثلثاً فيه: $AB = 5$، $AC = 7$، وقيس الزاوية $\\hat{A} = 60^\\circ$. لتكن $I$ منتصف القطعة $[BC]$. احسب الطول $BC$ ثم استنتج طول المتوسط $AI$."

  - tableLines:
      dir: rtl
      lines:
        - - th: "المرحلة الحسابية"
          - th: "خطوات التعويض والحساب"
          - th: "النتيجة الهندسية"
        - - th: "1. حساب الطول $BC$"
          - td: "$BC^2 = AB^2 + AC^2 - 2(AB)(AC)\\cos(\\hat{A})$<br>$BC^2 = 5^2 + 7^2 - 2(5)(7)\\cos(60^\\circ)$<br>$BC^2 = 25 + 49 - 70(0.5) = 74 - 35 = 39$"
          - td: "$BC = \\sqrt{39}$"
        - - th: "2. حساب المتوسط $AI$"
          - td: "$AB^2 + AC^2 = 2AI^2 + \\frac{BC^2}{2}$<br>$5^2 + 7^2 = 2AI^2 + \\frac{39}{2} \\implies 74 = 2AI^2 + 19.5$<br>$2AI^2 = 74 - 19.5 = 54.5 \\implies AI^2 = 27.25$"
          - td: "$AI = \\sqrt{27.25}$"
