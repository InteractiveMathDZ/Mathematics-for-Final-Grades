  - h3: "4. مسألة إدماجية موجهة (معايرة المقادير الحركية)"
    id: "final-problem"
  - parags:
      - rtl: "<b>وضعية الفحص:</b> يتحرك نقطي مادي صلب معلم مستقيم بتسارع ثابت مقداره النظري: $a(t) = 2$ (مقدر بوحدة الحقل الدولي $m \\cdot s^{-2}$). عيّن المعادلة الزمنية للسرعة $v(t)$، علماً أن السرعة الابتدائية المقاسة عند اللحظة الصفرية المرجعية تحقق القيد: $v(0) = 5$."
      - rtl: "<b>المسار التحليلي الحصيل:</b>"
      - rtl: "1. بإخضاع تابع التسارع الثابت للمكاملة العكسية، تتشكل الصيغة العامة للسرعة الحركية:"
      - center: "$v(t) = 2t + c \\quad (c \\in \\mathbb{R})$"
      - rtl: "2. بإدراج إحداثيات الشرط الابتدائي المقيد ($v(0) = 5$) صلب البنية المستخرجة، نحدد بدقة القيمة العددية للثابت السلمي:"
      - center: "$2(0) + c = 5 \\implies c = 5$"
      - rtl: "وعليه تتحدد المعادلة الزمنية الوحيدة للسرعة صلب هذا النظام وفق الدستور التالي:"
      - center: "$ v(t) = 2t + 5 $"
