elements:
  - h3: "المبحث 1: المفهوم والرتبة"
    id: "diff-concept"

  - parags:
      - rtl: "المعادلة التفاضلية ليست كمعادلات المتوسط ($x+5=0$) حيث المجهول عدد، بل هي معادلة المجهول فيها هو <b>دالة $y=f(x)$.</b>"
      - rtl: "تتميز هذه المعادلة بوجود مشتقات للدالة ($y', y'', ...$). <b>رتبة</b> المعادلة هي أعلى مشتقة تظهر فيها."
      - rtl: "<b>الحل العام:</b> هو مجموعة كل الدوال التي تحقق المعادلة (تحتوي دائماً على ثوابت $C$)."
      - rtl: "<b>الحل الخاص:</b> هو الدالة الوحيدة التي تحقق المعادلة مع 'شرط ابتدائي' معين (مثلاً عند $x=0$ تكون $y=1$)."

  - h3: "المبحث 2: الحل المباشر y' = f(x)"
    id: "direct-solution"

  - parags:
      - rtl: "هذا هو أبسط أنواع المعادلات التفاضلية، وحلها هو العودة المباشرة للدوال الأصلية التي درسناها سابقاً."
      - center: "$ y' = f(x) \\implies y = \\int f(x) dx + C $"



  - h4: "مثال تطبيقي (تثبيت المفهوم):"
    parags:
      - rtl: "حل المعادلة $y' = 2x + 1$ مع الشرط الابتدائي $y(0) = 3$."
      - rtl: "1. <b>الحل العام:</b> نكامل الطرفين $\\implies y = x^2 + x + C$."
      - rtl: "2. <b>إيجاد الثابت:</b> نعوض: "
      - center: "$x=0$ و $y=3$"
      - rtl: "في الحل العام:"
      - center: "$0^2 + 0 + C = 3 \\implies C = 3$."
      - rtl: "3. <b>الحل الخاص:<b> الدالة المطلوبة هي"
      - center: "$f(x) = x^2 + x + 3$."

        - rtl: "في البكالوريا، الشرط الابتدائي هو الذي يمنحك الحل الوحيد..."
