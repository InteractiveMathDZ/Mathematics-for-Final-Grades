  - h3: "1. الاستدلال التحليلي للحل المباشر"
    id: "direct-logic"

  - parags:
      - rtl: "يُمثل هذا النمط التفاضلي حلقة الوصل البنيوية المباشرة بين مفهومي التفاضل العكسي والدوال الأصلية؛ فعندما تقترن المشتقة الأولى لتابع مجهول بدالة معلومة ومتصلة $f(x)$، يؤول بروتوكول الحل مباشرة إلى إجراء مكاملة غير محددة للطرفين لاستعادة العبارة الدالية الأصلية:"
      - center: "$ y' = f(x) \\implies y = F(x) + c \\quad (c \\in \\mathbb{R}) $"
      - rtl: "حيث يُشير الرمز $F$ إلى دالة أصلية مفردة للتابع $f$ صلب مجال معالجته."

  - h3: "2. البروتوكول الإجرائي للتنفيذ والمعايرة"
    id: "execution-steps"

          - td: "تعيين الدالة الأصلية الإجمالية للعبارة $f(x)$ مع تدوين حد التعدد الجبري الحاصل (ثابت المكاملة $c$)."
        - - td: "2. موازنة الشرط الحدّي"

  - h3: "3. تطبيق تحليلي نموذجي معزز بشرط حدي"
    id: "detailed-example"
  - parags:
      - rtl: "<b>المسألة:</b> أوجد الحل الخاص للمعادلة التفاضلية الآتية: $y' = e^x + \\cos(x)$، علماً أن المنحنى الممثل للتابع يمر حتماً بمبدأ المعلم $(0,0)$."
      - rtl: "<b>تعيين الحل العام الشامل:</b> بالارتداد الاشتقاقي وإيجاد الدوال الأصلية المرجعية للحدود، نتحصل على عائلة الدوال:"
      - center: "$ y = e^x + \\sin(x) + c \\quad (c \\in \\mathbb{R}) $"
      - rtl: "<b>المعايرة العددية لثابت التعدد $c$:</b> بموجب القيد الحدي المقرن بمبدأ المعلم ($y(0) = 0$)، نُفعل التعويض العددي الآتي:"
      - center: "$ e^0 + \\sin(0) + c = 0 \\implies 1 + 0 + c = 0 \\implies c = -1 $"
      - rtl: "<b>صياغة الحل الخاص المقيد:</b> يؤول الحل الخاص والمعزول نظاميّاً إلى الدستور الدالي الآتي: $y = e^x + \\sin(x) - 1$."
