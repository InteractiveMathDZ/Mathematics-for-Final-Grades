  - parags:
      - rtl: "تُمثل هذه المعادلة التفاضلية الامتداد الخطّي التآلفي (Affine) للنموذج المتجانس السابق؛ حيث يتدخل الحد الثابت $b$ كمعامل إزاحة جبري مستمر. ويخضع السلوك التقاربي لحزمة التوابع الحصيلة نظاميّاً للموازنة التحليلية المشتركة بين الرتابة الأسية للمتغير والقيمة المحددة الناتجة عن المعاملات."

  - h3: "2. المنطوق النظري والمبرهنة المرجعية للحل العام"
    id: "theorem-affine"

  - parags:
      - rtl: "تقضي المبرهنة الرياضية بأن الحلول العامة للمعادلة التفاضلية الخطية التآلفية من الرتبة الأولى $y' = ay + b$ (شريطة كون المعامل السلمي $a \\neq 0$) هي عائلة الدوال الحقيقية المعرفة على مجموعة الأعداد الحقيقية $\\mathbb{R}$ وفق الدستور القياسي الآتي:"
      - center: "$ f(x) = C \\cdot e^{ax} - \\frac{b}{a} \\quad (C \\in \\mathbb{R}) $"
      - rtl: "يتضح تحليليّاً انقسام بنية التابع إلى شقين متمايزين: مركب أسي حركي متغير القيمة، وحد ثابت بنيوي معزول قيمته الصارمة $-\\frac{b}{a}$."

  - h3: "3. البروتوكول الخوارزمي لمعالجة النظم التآلفية"
    id: "protocol-affine"
  - tableLines:

          - td: "تحديد القيمة العددية الدقيقة لكل من المعاملين الحقيقيين $a$ و $b$ مع مراعاة الإشارات الجبرية."
        - - td: "2. التدوين الدستوري"
          - td: "صياغة العبارة الشاملة للحل العام مباشرة بالاستناد إلى القالب المرجعي المعاير: $y = C e^{ax} - \\frac{b}{a}$."
        - - td: "3. تعيين حد التعدد"
          - td: "إسقاط قيد الشروط الابتدائية للنظام لعزل قيمة الثابت الحقيقي $C$ وتحديد الحل الخاص."

  - h3: "4. تطبيق تحليلي نموذجي (معايرة النسق التآلفي المتناقص)"
    id: "lab-example"
  - parags:
      - rtl: "<b>المسألة:</b> حدد الحل الخاص للمعادلة التفاضلية المعطاة بالدستور: $y' = -2y + 4$، بشرط استيفاء القيد الابتدائي: $y(0) = 1$."
      - rtl: "<b>تعيين المعاملات وحساب حد الإزاحة:</b> بالمطابقة البنيوية المباشرة، يستقر التعيين عند: $a = -2$ و $b = 4$. ويؤول حساب حد التقارب الثابت صراحة إلى القيمة الحصيلة: $-\\frac{b}{a} = -\\frac{4}{-2} = 2$."
      - rtl: "<b>صياغة العبارة العامة للحل:</b> يستقر الدستور الشامل لعائلة الدوال عند الصيغة: $y = C e^{-2x} + 2$."
      - rtl: "<b>المعايرة العددية للثابت $C$:</b> بتفعيل القيد الابتدائي المقترن باللحظة صفر ($x=0 \\implies y=1$)، نتحصل على التكافؤ الجبري الآتي:"
      - center: "$ 1 = C \\cdot e^{-2(0)} + 2 \\implies 1 = C \\cdot 1 + 2 \\implies C = 1 - 2 = -1 $"
      - rtl: "<b>صياغة الحل الخاص المقيد:</b> يؤول الدستور الدالي المعزول والنهائي للنسق إلى العبارة المستقيمة الآتية: $f(x) = -e^{-2x} + 2$."

  - NB:
      title: "قيود ومحددات الاستساق التحليلي والفيزيائي:"
      type: "warning"
      NBs:
        - rtl: "<b>التأويل الفيزيائي لحد الاستقرار المرجعي:</b> تُمثل القيمة العددية المستقرة $-\\frac{b}{a}$ صلب النمذجة الفيزيائية للمسائل المقدار الحدي الثابت لـ 'النظام الدائم' (Steady State)؛ وهي القيمة التقاربية النهائية التي تؤول إليها الحالة الديناميكية للنسق بعد انقضاء فترة زمنية طويلة كافية لتلاشي المركب الأسي ($t \\to +\\infty$ عند $a < 0$)."
        - rtl: "<b>قيد التناظر الإشاري صلب الدستور:</b> يُلزم الطالب منهجيّاً بالانتباه الصارم للإشارة السالبة الأصلية المرافقة للكسر $-\\frac{b}{a}$ صلب نص المبرهنة، وإدراج إشارة المعامل المرجعي $a$ صلب المقام بصفة مستقلة منعا لحدوث مغالطات الإزاحة العمودية للبيان."
