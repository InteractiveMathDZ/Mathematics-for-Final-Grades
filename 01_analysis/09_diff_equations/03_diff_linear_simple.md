  - h3: "2. المنطوق النظري والمبرهنة المرجعية للحل العام"
    id: "theorem-ay"

  - parags:
      - rtl: "تقضي المبرهنة الرياضية بأن الحلول العامة للمعادلة التفاضلية الخطية المتجانسة $y' = ay$ على مجال الأعداد الحقيقية $\\mathbb{R}$ هي عائلة الدوال المعرفة وفق الدستور النمطي الآتي:"
      - center: "$ f(x) = C \\cdot e^{ax} \\quad (C \\in \\mathbb{R}) $"
      - rtl: "حيث يُمثل الرمز $C$ حد التعدد الجبري (ثابت حقيقي اختياري) ويتم تعيين قيمته الصارمة بالاستناد إلى قيد الشروط الابتدائية للنظام."

  - h3: "3. البروتوكول الخوارزمي لمعالجة المنظومة التفاضلية"

        - - td: "1. تعيين المعامل"
          - td: "استخراج القيمة الحقيقية الصارمة للمؤثر السلمي $a$ المقترن بالتابع $y$."
        - - td: "2. الصياغة الدستورية"
          - td: "تدوين العبارة الشاملة للحل العام مباشرة وفق القالب الأس القياسي: $y = C e^{ax}$."
        - - td: "3. موازنة القيد"
          - td: "تفعيل الشرط الابتدائي المعطى لحساب القيمة العددية الوحيدة للثابت $C$ وعزل الحل الخاص."

  - h3: "4. تطبيق تحليلي نموذجي (معايرة النسق الأسي)"
    id: "example-ay"
  - parags:
      - rtl: "<b>المسألة:</b> حدد الحل الخاص والمعزول نظاميّاً للمعادلة التفاضلية الآتية: $y' = 3y$، بشرط استيفاء القيد الابتدائي المقرن بالقيمة: $y(0) = 5$."
      - rtl: "<b>تعيين العبارة العامة:</b> بمطابقة البنية نجد معامل التناسب السلمي $a = 3$، مما يؤول مباشرة إلى صياغة عائلة الحلول: $y = C e^{3x}$."
      - rtl: "<b>المعايرة العددية لثابت التعدد $C$:</b> بالتعويض الإحداثي للشرط الابتدائي ($x=0 \\implies y=5$)، نتحصل على التكافؤ الآتي:"
      - center: "$ 5 = C \\cdot e^{3(0)} \\implies 5 = C \\cdot 1 \\implies C = 5 $"
      - rtl: "<b>صياغة الحل الخاص المقيد:</b> يستقر الدستور الدالي النهائي للنسق عند العبارة الحصيلة الصارمة: $f(x) = 5e^{3x}$."
