elements:
  - h3: "1. جوهر المعادلة"
    id: "essence-ay"

  - parags:
      - rtl: "في هذا النوع، سرعة تغير الدالة ($y'$) تتناسب طردياً مع قيمة الدالة ($y$) بمقدار ثابت $a$."
      - rtl: "الدالة الوحيدة في الرياضيات التي تحقق هذا الشرط (مشتقتها تشبهها) هي <b>الدالة الأسية</b>."

  - h3: "2. المبرهنة والحل العام"
    id: "theorem-ay"

  - parags:
      - rtl: "الحلول العامة للمعادلة التفاضلية $y' = ay$ على المجال $\\mathbb{R}$ هي مجموعة الدوال المعرفة بـ:"
      - center: "$ f(x) = C \\cdot e^{ax} $"
      - rtl: "حيث $C$ ثابت حقيقي يتم تحديده بواسطة الشروط الابتدائية."

  - h3: "3. بروتوكول الحل"
    id: "solving-protocol"
  - tableLines:
      dir: rtl
      lines:
        - - th: "المرحلة"
          - th: "الإجراء العملي"
        - - td: "1. التحديد"
          - td: "استخراج قيمة المعامل $a$ من المعادلة."
        - - td: "2. كتابة الحل"
          - td: "صياغة الحل العام مباشرة: $y = C e^{ax}$."
        - - td: "3. التثبيت"
          - td: "تعويض الشرط الابتدائي لإيجاد قيمة $C$."

  - h3: "4. مثال تطبيقي"
    id: "example-ay"
  - parags:
      - rtl: "<b>المسألة:</b> أوجد حل المعادلة $y' = 3y$ بحيث يكون $y(0) = 5$."
      - rtl: "<b>الحل العام:</b> بما أن $a = 3$، فإن الحل هو $y = C e^{3x}$."
      - rtl: "<b>إيجاد C:</b> نعوض $x=0 \\to y=5$، نجد $5 = C e^{3(0)} \\implies 5 = C \\cdot 1 \\implies C = 5$."
      - rtl: "<b>الحل الخاص:</b> $f(x) = 5e^{3x}$."
