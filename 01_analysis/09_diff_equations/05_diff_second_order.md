  - h3: "1. المقاربة التحليلية والميكانيكية للرتبة الثانية"
    id: "vibration-concept"

  - parags:
      - rtl: "تُمثل هذه الصيغة النموذج التحليلي القياسي للمعادلات التفاضلية الخطية المتجانسة من الرتبة الثانية ذات المعاملات الثابتة. تصف المعادلة ديناميكية النظم المحافظة (Conservative Systems) حيث يتناسب القيد التسارعي (المشتق الثاني $y''$) طرديّاً ويعاكس في الاتجاه مقدار الإزاحة الخطية للتابع $y$. ويضمن التربيع السلمي للمؤثر $\\omega^2$ اتصاف السلوك الحصيلي بالدورية الجيبية والابتعاد البنيوي عن الرتابة الأسية."

  - h3: "2. المنطوق النظري والمبرهنة المرجعية للحل الشامل"
    id: "theorem-second-order"

  - parags:
      - rtl: "تقضي المبرهنة الرياضية بأن الحلول العامة للمعادلة التفاضلية من الرتبة الثانية $y'' + \\omega^2y = 0$ (شريطة كون النبض السلمي $\\omega \\neq 0$) هي عائلة الدوال الحقيقية المعرفة والقابلة للاشتقاق مرتين على $\\mathbb{R}$ وفق التشكيل الخطي الآتي:"
      - center: "$ f(x) = A \\cos(\\omega x) + B \\sin(\\omega x) \\quad (A, B \\in \\mathbb{R}) $"
      - rtl: "ويمكن صياغتها بالصيغة التوافقية المدمجة المكافئة:"
      - center: "$ f(x) = y_{\\max} \\cos(\\omega x + \\phi) $"
      - rtl: "حيث تُمثل المقادير ($A, B$) أو ($y_{\\max}, \\phi$) ثوابت هندسية حقيقية، يتطلب تعيين قيمها الصارمة وعزل الحل الخاص إخضاع المنظومة لقيدين ابتدائيين متزامنين (رتبة الموضع ورتبة السرعة التفاضلية عند اللحظة الابتدائية)."

  - h3: "3. البروتوكول الخوارزمي لمعالجة النظم الاهتزازية"

  - h3: "4. تطبيق تحليلي نموذجي (معايرة النسق الجيبي النقّي)"
    id: "vibration-example"
  - parags:
      - rtl: "<b>المسألة:</b> حدد الحل الخاص للمعادلة التفاضلية من الرتبة الثانية: $y'' + 4y = 0$، بشرط استيفاء القيدين الابتدائيين المتزامنين: $y(0) = 1$ و $y'(0) = 0$."
      - rtl: "<b>تعيين النبض الذاتي وصياغة العبارة العامة:</b> بمطابقة العبارة نجد أن: $\\omega^2 = 4 \\implies \\omega = 2$. وعليه يستقر الدستور الشامل لعائلة الحلول عند الصيغة: $y(x) = A \\cos(2x) + B \\sin(2x)$."
      - rtl: "<b>تفعيل القيد الابتدائي الأول:</b> بالتعويض المباشر في دالة الموضع ($x=0 \\implies y=1$):"
      - center: "$ A \\cos(0) + B \\sin(0) = 1 \\implies A \\cdot(1) + B \\cdot(0) = 1 \\implies A = 1 $"
      - rtl: "<b>تفعيل القيد الابتدائي الثاني:</b> يستلزم البروتوكول إجراء الاشتقاق الفوري للعبارة العامة: $y'(x) = -2A \\sin(2x) + 2B \\cos(2x)$. وبتعويض شرط السرعة الابتدائية ($x=0 \\implies y'=0$):"
      - center: "$ -2(1) \\sin(0) + 2B \\cos(0) = 0 \\implies 0 + 2B \\cdot(1) = 0 \\implies B = 0 $"
      - rtl: "<b>صياغة الحل الخاص المقيد:</b> بإسقاط القيم العددية للثوابت، يؤول الدستور الدالي المعزول والنهائي للنسق إلى العبارة الموجية الصارمة: $f(x) = \\cos(2x)$."

  - NB:
      title: "قيود ومحددات الاتساق البنيوي صلب الرتبة الثانية:"
      type: "danger"
      NBs:
        - rtl: "<b>شرط الاتساق الإشاري للمؤثر:</b> يُقيد تفعيل مبرهنة الحل الجيبي الدوري بالتحقق الصارم من الإشارة الموجبة الفاصلة بين حدي المعادلة ($y'' + \\omega^2y = 0$). ففي حال ظهور إشارة سالبة ($y'' - \\omega^2y = 0$)، يختل السلوك الدوري نظاميّاً وتؤول الحلول حتماً نحو الفضاء الأسي الحركي (الدوال الأسية أو الزائدية)."
        - rtl: "<b>المعايرة الزمنية للمقادير الفيزيائية:</b> يُمثل المعامل المعزول $\\omega$ صلب الأدبيات الفيزيائية المعتمدة المقدار المميز لـ 'النبض الذاتي' (Natural Angular Frequency) للنظام الاهتزازي، ومنه يُشتق الدستور التحليلي للدور القياسي للحركة عبر العلاقة الطوبولوجية العكسية: $T = \\frac{2\\pi}{\\omega}$."
